Support: Point of action

Observations
  • Support for any data related tasks helps for adoption.
  • Groups and individual researchers need customized help.
  • Individual RDM support fosters culture-change.
Conclusions
  • Sound data-management practices are a prerequisite for and start long before data archiving/publishing.
  • Intervention directly where data is produced seems indispensable to achieve good data-management practices.

Perceived effort for data preparation is a main barrier for archiving & publishing [1,2,3].

ERIC/open

Since November 1, 2019: https://opendata.eawag.ch 🔗
  • Same system. But completely independent.
    (Webapp, SOLR, Postgres, Storage)
  • No ingress, no users.
  • Completely dockerized.
  • In the process of being populated (~80 packages backlog).
  • Every package has a DOI (DataCite).
